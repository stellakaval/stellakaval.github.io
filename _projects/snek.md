## Overview 
The project involves implementing a snake game in C. Players control snakes on a grid, navigating to eat fruits while avoiding walls and collisions with other snakes.

## Description
1. **create_default_state**: Initialize a default game state.
2. **free_state**: Free memory allocated for the game state.
3. **print_board**: Print the game board to a file.
4. **update_state**: Implement game logic to update the state.
5. **load_board**: Read a game board from a file.
6. **initialize_snake**: Initialize snake structs based on the loaded board.
7. **main**: Update the board by one time step.
8. **README**: Provide feedback and reflections on the project.


### Skills Developed:
- Allocating and freeing memory.
- Reading and printing game boards.
- Implementing core game rules.
- Testing and debugging procedures.
- Writing specific task functions.
- Accessing and modifying struct fields.
