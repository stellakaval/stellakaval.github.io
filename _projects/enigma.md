I worked on a project where I created a simulation of the Enigma machine in Java. The Enigma machine was a mechanical device used to encrypt and decrypt messages during World War II, and my program allowed users to input messages and configure the machine with different rotors and reflectors in order to encrypt and decrypt those messages. I implemented a series of rotors and a reflector to create a progressive substitution cipher, using a one-to-one mapping of the alphabet onto itself to replace each letter in the message with another letter based on the current configuration of the machine. While working on this project, I developed my programming skills in several areas. I implemented the Enigma machine's substitution cipher algorithm using arrays and lists to store and manipulate the rotors and reflectors. I also designed and implemented classes to represent the different components of the Enigma machine, including the rotors, reflector, and Enigma machine itself. These classes used object-oriented programming techniques, such as encapsulation and inheritance, to represent the relationships between the different components and to allow for reuse and customization of the Enigma machine's functionality.
