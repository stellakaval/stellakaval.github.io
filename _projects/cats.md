The goal of this project was to create a program that can measure typing speed and correct spelling errors. I wrote code in Python and edited a file "cats.py" file. The project included text samples to be typed and a web server for a graphical user interface, provided by UC Berkeley’s CS61A course staff. I worked with files and strings, so I needed to know how to read and write to files and manipulate strings. As I worked on the project, I ran into bugs and errors, so it was important for me to know how to debug my code. It was also crucial for me to test my code frequently to make sure everything was working correctly. I used the "ok" autograder to test my code and track my progress as I worked on the project.
