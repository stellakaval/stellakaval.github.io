The purpose of the Hog project is to create a simulator and strategies for a dice game called Hog. Players take turns rolling dice and trying to be the first to reach a certain number of points. There are special rules for rolling zero dice and reaching perfect square scores. By completing the Hog project, I learned several programming skills in Python, including how to use control statements and higher-order functions. I also learned how to design and implement a simulation for a dice game. I practiced debugging and testing my code using the "ok" autograder. I also had the opportunity to work with a partner, which allowed me to practice teamwork and collaboration skills.
