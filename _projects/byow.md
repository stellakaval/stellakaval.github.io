## Overview

"Build Your Own World" is a video game focused on creating a 2D tile-based world exploration engine.

## Description

- [Project Specification](https://inst.eecs.berkeley.edu/~cs61b/sp22/materials/proj/proj2/index.html)
- This project involves designing and implementing a system capable of generating dynamic worlds on a 2D grid
- Key features include pseudo-random world generation, distinct rooms and hallways, and interactive gameplay elements

## Skills

- Algorithmic design for world generation
- Implementation of data structures for world representation
- Control flow and user input handling for interactive gameplay
- Project management and iterative development in software engineering
