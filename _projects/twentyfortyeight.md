## Introduction
Project 0: 2048 Game introduces students to Java programming and essential tools like IntelliJ IDE and JUnit for writing and running unit tests. The task involves implementing core logic for the popular 2048 game within the Model.java file.

## Description
- Implements the core logic of the 2048 game, focusing on four methods within the Model.java file.
- Students work with existing GUI code, handling key-press events and game state updates.
- The game involves a 4x4 grid where tiles with powers of 2 are merged by tilting the board in four directions.
- Points are earned when tiles merge, and the game ends when no more moves are possible or a tile with a value of 2048 is formed.
- Detailed rules and examples provided in the specification document.

## Skills
- Java programming fundamentals.
- Understanding of game logic and event handling.
- Utilization of existing codebase and libraries.
